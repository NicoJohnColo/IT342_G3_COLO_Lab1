# User Registration and Authentication App

## Project Description
This project is a **User Registration and Authentication system** that allows users to sign up, log in, and securely manage their accounts. It includes backend APIs, a web frontend, and optional mobile integration.

## Technologies Used
- **Backend:** Node.js, 
- **Database:** MongoDB 
- **Web App:** React 
- **Mobile App:** React Native/ Kotlin
- **Authentication:** JWT (JSON Web Tokens) or session-based auth
- **Version Control:** Git & GitHub
